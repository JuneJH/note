# 集合

> Collection

## 1. Collection

> 存储的都是value

#### 1. List

> 有序可重复

- ArrayList: 底层数组,适合遍历
- LinkedList: 底层链表,适合插入
- Vector: 是ArrayList集合的早期版本,线程同步(synchronized)
- Stack: 栈,先进后出`push`,`pop`,`peek`

**ArrayList 和 Vector的区别 线程安全**

#### 2.Set

> 无序无重复

#### 3. Queue

> 队列,先进先出

- offer
- peek
- poll



## 2. Map

> 存储的都是key-value
>
> key无序无重复,value无序可重复
