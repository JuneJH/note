# Linux 基本操作命令



## 基本操作

| 命令                                     | 作用                                   | 备注 |
| ---------------------------------------- | -------------------------------------- | ---- |
| ll                                       | 显示列表   ls -l 缩写                  |      |
| ls                                       | 横着展示  ls -al 看到隐藏文件          |      |
| xz -d                                    | 解压                                   |      |
| tar -xf                                  | 解压                                   |      |
| ln -s ~/目标位置 /usr/bin/指令名  软链接 | 软接链                                 |      |
| cd                                       | 进入                                   |      |
| mkdir                                    | 创建路径                               |      |
| Rmdir                                    | 删除一个空文件   rm -rf 删除本文件一切 |      |
