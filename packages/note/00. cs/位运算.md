# 位运算

> 计算机只能认识0和1



## 1. 基础概念

1. `<<`  左移一位，乘以2
2. `>>`  右移一位, 整除以2
3. `&`  两个同时是1的时候结果是1，否则0
4. `｜`  两个同时是0的时候结果是0，否则1
5. `^` 异或，两个不一样为1

## 实现一个加法

- 异或运算能实现未进位的加法结果
- 与运算能实现进位结果

```js
function add(a,b){
    if(a == 0){
        return b;
    }
    if(b == 0){
        return a;
    }

    return add(a ^ b,(a & b) << 1);
}
```
