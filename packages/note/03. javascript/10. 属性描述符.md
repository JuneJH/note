# 10. 属性描述符

> 作为 vue2 实现响应式基础只是其中一小场景

## 1.  得到一个对象的属性描述符号

1. 使用`Object.getOwnPropertyDescriptor`获取

   ```js
   const obj = {name:"123"};
   Object.getOwnPropertyDescriptor(obj,'name');
   // {value: '123', writable: true, enumerable: true, configurable: true}
   ```

2. `value`: 表示该对象的这个属性的值

3. `writable`: 是否可重写

4. `enumerable`： 是否可遍历

5. `configurable`: 属性描述配置是否可更改

## 2. 设置一个对象的属性描述符

1. 使用`Object.defineProperty`设置

   ```js
   const obj = {name:"123"};
   Object.defineProperty(obj,'name',{
     value:"june",
     writable:false, // 不可重写，不能修改
     enumberable:false, // 不可遍历
     configurable:false // 不可修改属性符配置
   })
   ```

2. 设置存储器

   ```js
   const obj = {name:"123"};
   Object.defineProperty(obj,'name',{
   	get:()=>{},
     set:()=>{}
     configurable:false // 不可修改属性符配置
   })
   ```
