# 算法

## 1. 计算思维

1. 分解
   - 把复杂问题拆解成一个一个简单问题，逐一击破
2. 模式
   - 模式识别就是在一组数据中找出特征或规则，用于对于数据进行分类，以作为决策判断的依据
3. 抽象
   - 提取问题关键部分，忽略无关紧要的细节部分
4. 算法
   - 解决问题的方法和步骤

## 2. 算法的条件

1. 输入
2. 输出
3. 明确性
4. 有限性
5. 有效性

## 3. 算法的描述

#### 1. 流程图

1. 椭圆：表示算法的开始和结束
2. 平行四边形：表示算法的输入和输出
3. 矩形：表示处理的步骤
4. 菱形：表示条件或者分支
5. 流程线： 连接

#### 2. 伪代码

```pseudocode
INPUT "输入一个值"
OUTPUT "输出一个值"

Counter <- 变量

IF A > B
	THEN
		OUTPUT "a 大于 b"
	ELSE
		OUTPUT "B 大于 A"
 ENDIF
 
 FOR COUNTER <- 1 TO 10
 	OUTPUT "循环"
 NEXT Counter
```
