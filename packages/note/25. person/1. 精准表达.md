# 1. 精准表达



## 1. 学会倾听

1. 是否是为了说而听
   1. 对话还在说话，你已经开始打腹稿，准备自己的意见
   2. 对话话音还没落，你就迫不及待地发表自己的观点
   3.  在大多数对话时间，都是你说的多，多方说的少
   4. 你说的已经足够多了，但结果常常
2. 机关枪倾听者的信念
   1. 你是错的
   2. 我是对的
3. 非合作思维
4. 标签
   1. 打上标签
   2. 以自我为中心



### 有效的倾听方法

1. 反应式倾听
   1. 忍：搁置自己的兴趣（想象有一束光，打在对方头上，四周黑暗，他是我的核心）
   2. 等：想插话、打断的时候在等一分钟（避免特别快给人贴标签，以己度人）
   3. 问：通过提问，进一步明确是不是真的听懂了对话的话
      1. 复述式提问
      2. 开发式提问
2. 行动式倾听
