#  CSS 基础

> **CSS（Cascading Style Sheets）**：层叠样式表，用于描述 **HTML 文档的样式与表现**。

------

## 🧩 一、CSS 基础结构

### 🔹 1. CSS 声明块（Declaration Block）

```css
div {
  color: red;
}
```

- **选择器（selector）**：`div`，用于选中元素。
- **声明块（declaration block）**：花括号 `{}` 内的部分。
- **声明（declaration）**：`color: red;`，由属性和值组成。

------

### 🔹 2. CSS 的三种书写方式

#### 🏠 内部样式表（Internal Style Sheet）

```css
<head>
  <style>
    p { color: blue; }
  </style>
</head>
```

> 写在 `<style>` 标签中，一般放在 `<head>` 内能够先加载。

------

#### 💡 内联样式（Inline Style）

```css
<p style="color: red;">Hello CSS</p>
```

> 直接写在 HTML 标签的 `style` 属性中，权重相对较高但不便复用。

------

#### 📄 外部样式表（External Style Sheet）

```css
<link rel="stylesheet" href="style.css" />
```

**优点：**

- 样式可复用；
- 浏览器可缓存；
- 结构与样式分离，易于维护。

------

## 🖋️ 二、常见样式属性

### 🎨 字体与文字属性

- **三原色，色值** ：光学三原色（红，绿，蓝），每个颜色可以使用0~255之间的数字来表示，色值
- rgb()/rgba()多的一个值表示透明通道
- hex(16进制)表示法：#红绿蓝
  1. 淘宝红：#f40
  2. 黑色：#000
- 每个元素必须有字体大小，如果没有声明，则直接使用父元素大小。父元素如果也没有，就采用基准大小（可以在浏览器上设置）

| 属性              | 作用         | 示例                                |
| ----------------- | ------------ | ----------------------------------- |
| `color`           | 文本颜色     | `color: #333;`                      |
| `font-size`       | 字体大小     | `font-size: 16px;`                  |
| `font-weight`     | 粗细         | `font-weight: bold;`                |
| `font-family`     | 字体         | `font-family: "Arial", sans-serif;` |
| `font-style`      | 斜体         | `font-style: italic;`               |
| `text-decoration` | 下划线等修饰 | `text-decoration: underline;`       |
| `text-align`      | 对齐方式     | `text-align: center;`               |
| `text-indent`     | 首行缩进     | `text-indent: 2em;`                 |
| `line-height`     | 行高         | `line-height: 1.5;`                 |

------

### 📦 尺寸与盒模型属性

| 属性               | 说明                                     |
| ------------------ | ---------------------------------------- |
| `width` / `height` | 元素宽高                                 |
| `display`          | 盒子类型（`block`、`inline`、`flex` 等） |
| `visibility`       | 元素可见性                               |
| `vertical-align`   | 行内元素垂直对齐方式                     |

------

### 🌈 背景相关属性

| 属性                    | 说明     | 示例                             |
| ----------------------- | -------- | -------------------------------- |
| `background-color`      | 背景色   | `background-color: #fafafa;`     |
| `background-image`      | 背景图   | `background-image: url(bg.png);` |
| `background-repeat`     | 平铺     | `background-repeat: no-repeat;`  |
| `background-size`       | 背景大小 | `background-size: cover;`        |
| `background-position`   | 背景位置 | `background-position: center;`   |
| `background-attachment` | 滚动方式 | `background-attachment: fixed;`  |

📘 **快捷写法(复合属性)：**

```
background: url(bg.jpg) no-repeat center / cover;
```

## 🧮 三、继承与属性值计算

💡 **CSS 属性值的计算过程：**

1. 确定声明值（根据样式表）
2. 层叠冲突（权重、优先级）
3. 继承父元素值
4. 应用默认值（如无设置）

🔸 特殊值：

- `inherit`：继承父级
- `initial`：恢复为默认值

------

## 🧱 四、布局模型（Visual Formatting Model）

CSS 布局主要分为三类：

### 1️⃣ 常规流（Normal Flow）

- 默认布局模式；
- 块级元素独占一行；
- 行内元素水平排列；
- 上下外边距会**合并（collapse）**。

📘 **居中技巧：**

```css
div {
  width: 300px;
  margin: 0 auto;
}
```

------

### 2️⃣ 浮动布局（Float）

```css
float: left | right | none;
```

🔹 特点：

- 元素脱离文档流；
- 会影响文字环绕；
- 父级高度坍塌。

💡 **清除浮动：**

```css
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

------

### 3️⃣ 定位布局（Position）

| 属性       | 特点                           |
| ---------- | ------------------------------ |
| `static`   | 默认值                         |
| `relative` | 相对定位（不脱流）             |
| `absolute` | 绝对定位（相对最近的定位祖先） |
| `fixed`    | 固定定位（相对视口）           |
| `sticky`   | 吸顶定位                       |

📘 **定位居中：**

```css
position: absolute;
top: 0; bottom: 0; left: 0; right: 0;
margin: auto;
```

------

### 📊 层叠上下文（Stacking Context）

层叠顺序从低到高：

1. 背景与边框
2. 负 `z-index`
3. 普通流元素
4. 浮动元素
5. 定位元素（`z-index: auto`）
6. 定位元素（`z-index > 0`）

------

## 🧾 五、常见问题汇总

### 1️⃣ 图片底部留白

原因：`img` 是行内元素，与基线对齐。
 解决方案：

```css
img { display: block; }
```

或

```css
.parent { font-size: 0; }
```

------

### 2️⃣ 字体与行高关系

```css
font-size：文字高度（content area）
line-height：行高（virtual area）
```

行高过小会导致文字重叠。

------

## ⚡ 六、渐进增强 vs 优雅降级

| 思路                                   | 定义                         | 应用                 |
| -------------------------------------- | ---------------------------- | -------------------- |
| **渐进增强 (Progressive Enhancement)** | 先实现核心功能，再增强新特性 | 优先兼容所有浏览器   |
| **优雅降级 (Graceful Degradation)**    | 先完成完整功能，再做兼容     | 对旧浏览器做兼容降级 |

------

## ✅ 七、总结

✨ **CSS 核心思想：**

- 层叠 + 继承 + 盒模型
- 常规流 + 浮动 + 定位三大布局
- 兼顾语义化与兼容性
- 遵循“渐进增强”的设计理念
