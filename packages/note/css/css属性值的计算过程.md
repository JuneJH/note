# CSS属性值的计算过程

> 层叠、继承等相关规则，一个HTML元素如果需要被渲染出来那么它的的css属性必须全部有值

## 1. 确定声明值

1. 所有样式表中没有冲突的声明，直接作为css属性值
2. 比如作者样式表中和浏览器默认样式表中有针对同一标签进行设置相同属性，该属性在这一步不做处理，只处理单一属性

## 2. 层叠冲突

1. 对样式表有冲突的声明使用层叠规则，确定css属性
2. 比较重要性 作者样式表 优先 浏览器默认样式表 **此时不考虑权重**
3. 权重占比
4. 比较源次序

## 3. 使用继承

1. 对仍然没有值的属性，若可以继承，则继承父元素的值（一般文字属性可继承）

## 4. 使用默认值

1. 对仍然没有值的属性，使用默认值



## 5. 常见疑惑

1. 为什么对a标签的包含块设置了颜色对a标签并不生效？
   1. 因为浏览器默认样式表中对a设置了颜色
   2. 可以在全局样式表中设置```a:{color:inherit}```

1. 为什么我写```*{margin:0;padding:0}```会覆盖浏览器默认样式表中的```body{margin:8px}```?
   1. 相比权重占比之前还有一个规则，比较重要性，作者样式表优先浏览器默认样式表
